## Why it's different

<div className="not-prose my-6 grid gap-3 sm:grid-cols-2">
  <div className="rounded-xl border border-fd-border bg-fd-card p-4">
    <div className="text-xs font-medium uppercase tracking-wider text-fd-muted-foreground">Shape-first</div>
    <p className="mt-2 text-sm">Seed an initial value; the prompt + JSON parser are derived automatically.</p>
  </div>
  <div className="rounded-xl border border-fd-border bg-fd-card p-4">
    <div className="text-xs font-medium uppercase tracking-wider text-fd-muted-foreground">Cancel-on-edit</div>
    <p className="mt-2 text-sm">If the user calls `setValue` mid-generate, the in-flight request is aborted. Manual intent always wins.</p>
  </div>
  <div className="rounded-xl border border-fd-border bg-fd-card p-4">
    <div className="text-xs font-medium uppercase tracking-wider text-fd-muted-foreground">Built-in cache</div>
    <p className="mt-2 text-sm">LRU(16) keyed by `(shape, context)` — repeated generates with the same context are free.</p>
  </div>
  <div className="rounded-xl border border-fd-border bg-fd-card p-4">
    <div className="text-xs font-medium uppercase tracking-wider text-fd-muted-foreground">JS-compatible</div>
    <p className="mt-2 text-sm">No generics required. The runtime shape comes from the seed, so plain JS callers get the same behavior.</p>
  </div>
</div>

## 1. Primitive — random number

The simplest case: a `number` initial value plus a context string.

### Skeleton

<Preview
  demo={<SmartStateRandomSkeleton />}
  source={`import { SmartProvider, useSmartState } from '@extedcoud/smart-components';
import { createMockClient } from '@extedcoud/smart-components/adapters/mock';

const client = createMockClient({
  complete: async () => String(Math.floor(Math.random() * 100) + 1),
});

function Demo() {
  const [n, setN, ai] = useSmartState(0, 'a random integer between 1 and 100');
  return (
    <div>
      <button onClick={() => ai.generate()} disabled={ai.status === 'loading'}>
        {ai.status === 'loading' ? 'Generating…' : 'Generate'}
      </button>
      <button onClick={() => setN(0)}>Reset</button>
      <pre>n = {n}</pre>
    </div>
  );
}

export default function Example() {
  return <SmartProvider client={client}><Demo /></SmartProvider>;
}`}
/>

## 3. Array — generated tags (with `options.shape`)

When `initial` is empty (`[]`, `null`, `undefined`), pass `options.shape` so the hook knows what to ask for.

### Skeleton

<Preview
  demo={<SmartStateTagsSkeleton />}
  source={`const [tags, , ai] = useSmartState<string[]>([], 'tags for a sci-fi blog post', {
  shape: { type: 'array', item: 'string' },
});

<button onClick={() => ai.generate()}>Generate tags</button>
<pre>{JSON.stringify(tags)}</pre>`}
/>

### Stylized

A category switcher feeding a per-call context override, rendering tags as gradient pills.

<Preview
  demo={<SmartStateTagsStylized />}
  source={`const CATEGORIES = [
  { key: 'scifi', label: '🛸 Sci-fi blog post', context: 'tags for a sci-fi blog post' },
  { key: 'recipe', label: '🍳 Weeknight recipe', context: 'tags for a weeknight recipe post' },
  { key: 'travel', label: '🌍 Travel guide', context: 'tags for a travel guide article' },
];

const [category, setCategory] = useState(CATEGORIES[0]);
const [tags, , ai] = useSmartState<string[]>([], category.context, {
  shape: { type: 'array', item: 'string' },
});

<div>
  {CATEGORIES.map((c) => <button key={c.key} onClick={() => setCategory(c)}>{c.label}</button>)}
</div>
<div>
  {tags.map((t) => <span className="rounded-full bg-emerald-500/15 px-3 py-1">#{t}</span>)}
</div>
<button onClick={() => ai.generate(category.context)}>Generate tags</button>`}
/>

## 4. Per-call context override

`ai.generate(contextOverride)` replaces the default context for one call — useful when the prompt is user-driven.

### Skeleton

<Preview
  demo={<SmartStateContextSkeleton />}
  source={`const [user, , ai] = useSmartState({ name: '', age: 0, bio: '' });
const [prompt, setPrompt] = useState('a brave wizard');

<input value={prompt} onChange={(e) => setPrompt(e.target.value)} />
<button onClick={() => ai.generate(prompt)}>Generate</button>
<pre>{JSON.stringify(user, null, 2)}</pre>`}
/>

### Stylized

A character-builder card with quick-pick preset chips and a soft amber/rose result panel.

<Preview
  demo={<SmartStateContextStylized />}
  source={`const PRESETS = [

  'a brave wizard from a forgotten realm',

const [user, , ai] = useSmartState({ name: '', age: 0, bio: '' });

<input value={prompt} onChange={(e) => setPrompt(e.target.value)} />
<button onClick={() => ai.generate(prompt)}>✦ Generate</button>

<div className="rounded-xl bg-gradient-to-br from-amber-500/5 to-rose-500/5 p-4">

</div>`}
/>
